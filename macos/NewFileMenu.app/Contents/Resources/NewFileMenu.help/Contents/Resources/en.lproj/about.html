About New File Menu

New File Menu allows you to create new files quickly via the Finder context menu.

New File Menu is the coolest way to create a new file on the Mac OS X platform. Simply right click in any Finder window and select the kind of document you would like to create!

Important: You need to enable "New File Menu Extension" in "System Preferences Â» Extensions" to make the context menu appear.
