Finder Context Menu Extension

New File Menu is implemented as a Finder extension, you need to configure it before using it.

  1. Enable the Finder context menu extension
    1. Open the System Preferences app and navigate to the Extensions section
    2. Enable the "New File Menu Extension" Finder extension
  2. Configure the locations you would like to add the context menu
    1. Launch New File Menu
    2. Select the "Finder" tab
    3. Add/remove the locations that you would like to add/remove the context menu
    4. You can also enable/disable an item to add/remove the context menu
